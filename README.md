## 📋 Proje Hakkında
Bu proje, kripto para yatırımcılarının en çok ihtiyaç duyduğu piyasa verilerini (fiyatlar, değişim oranları ve teknik grafikler) tek bir arayüzde birleştirmek amacıyla geliştirilmiştir. Karmaşık veri setlerini kullanıcı dostu bir dashboard yapısıyla sunar.

### ✨ Temel Özellikler
* **Canlı Grafik Entegrasyonu:** TradingView'in gelişmiş grafik araçları ile anlık fiyat takibi.
* **Piyasa Özeti:** En popüler kripto paraların performanslarını gösteren dinamik listeler.
* **Teknik Analiz Görünümü:** Seçili pariteler için hızlı teknik analiz göstergeleri.
* **Responsive Tasarım:** Hem masaüstü hem de mobil cihazlarla tam uyumlu arayüz.
* **Cyberpunk Estetiği:** Modern, neon detaylara sahip karanlık mod (Dark Mode) tasarımı.

---

## 🛠️ Kullanılan Teknolojiler
Proje geliştirilirken kullanılan temel teknolojiler ve kütüphaneler:

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Veri & Grafik:** [TradingView Widget API](https://www.tradingview.com/widget/) (Gerçek zamanlı piyasa verileri ve görselleştirme için)
* **Deployment:** GitHub Pages

## 🚀 Kurulum ve Yerel Çalıştırma

Projeyi kendi bilgisayarınızda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

1.  **Depoyu (Repository) Klonlayın:**
    ```bash
    git clone [https://github.com/emirhanuzgun/crypto-dashboard.git](https://github.com/emirhanuzgun/crypto-dashboard.git)
    ```

2.  **Proje Dizinine Gidin:**
    ```bash
    cd crypto-dashboard
    ```

3.  **Çalıştırın:**
    Proje sadece HTML/CSS/JS içerdiği için herhangi bir bağımlılık yüklemenize gerek yoktur. `index.html` dosyasını tarayıcınızda açarak veya bir **Live Server** (VS Code eklentisi gibi) kullanarak hemen görüntüleyebilirsiniz.
